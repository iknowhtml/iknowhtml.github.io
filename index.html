Perish

A RESTful API for anonymous and temporary file storage. Implemented in Rails, files are hashed using SHA256 and the digest is used as an unique ID for downloading the file. Files are also encrypted using AES256 and then stored on AWS S3. Winner of the First API | Summer Jam Hackathon!
