Mogura

An AWS-based cloud cryptocurrency mining operation using an EC2 Spot Fleet to mine. The mining P2 spot instances are managed as a Spot Fleet and the bid price is dynamically updated to order to insure profitability using the a Node application and the AWS SDK. Terraform is used to provision and configure the infrastructure. The project also leverages Packer to build AMIs preinstalled with necessary software for mining. In progress.

ManekiBoto

A cryptocurrency arbitrage bot for automating cryptocurrency trading between various markets. Uses a collection of custom API libraries for pulling public data, as well as programatically initiating trades on Poloniex and Kraken. Leverages a custom algorithm to determine maximum profitability between currency pairs and markets. Written entirely in Node.js. In progress.

Lambdas at UVA Website

The official home page of Lambda Phi Epsilon, Alpha Tau Chapter, at the University of Virginia! Designed with Sketch and written as an SPA with React and styled using CSS4 through PostCSS and CSSNext. Deployed to Github pages with Travis to automate the build. Still in progress, but check out the mockup!

Perish

A RESTful API for anonymous and temporary file storage. Implemented in Rails, files are hashed using SHA256 and the digest is used as an unique ID for downloading the file. Files are also encrypted using AES256 and then stored on AWS S3. Winner of the First API | Summer Jam Hackathon!
